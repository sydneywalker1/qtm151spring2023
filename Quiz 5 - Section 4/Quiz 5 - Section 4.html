(c) Compute growth rates

  • Calculate a new column "growth_cb" which computes the percentage daily
    growth rate of "cb_to_usd".

Hint: Use .diff() and .shift(1) as in Lecture 22

# Write your own code
bitcoin["diff_cb"] = bitcoin["cb_to_usd"].diff()
bitcoin["lag_cb"] = bitcoin["cb_to_usd"].shift(1)
bitcoin["growth_cb"] = (bitcoin["diff_cb"]/bitcoin["lag_cb"])*100

display(bitcoin["growth_cb"])

(d) Flag subset of data and plot

# Write your own code
bitcoin["bool_cond"] = (bitcoin["growth_cb"] > 15) | (bitcoin["growth_cb"] < -15)

plt.plot("date","growth_cb",
         data = bitcoin.sort_values("date", ascending = True))
plt.xlabel("Time")
plt.ylabel("Daily percentage change")
plt.title("Change in the CoinBase Bitcoin Exchange Rate")
vec_y = bitcoin["growth_cb"]
plt.fill_between(x=bitcoin["date"],
                 y1=vec_y.min(),
                 y2 = vec_y.max(),
                 where = bitcoin["bool_cond"],
                 alpha = 0.3, color = "red")
plt.show()

(e) Aggregate dataset by time period

    the date column and freq to "m" (month)
  • Follow the instructions in Lecture 21 to compute an aggregate dataset

# Write your own code
month_config = pd.Grouper(key="date", freq = "m")
monthly_data = (bitcoin
                .groupby(month_config)
                .agg(cb_to_usd = ("cb_to_usd","std"))
                .reset_index())
